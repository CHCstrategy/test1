# Australia Capital Growth Suburb Finder

A lightweight chat-style web app that lets a user describe the type of Australian property investment opportunity they want and returns ten suburb suggestions for capital growth.

## Features

- Chat interface with a free-form prompt box.
- Generates ten suburb recommendations tailored to keywords such as state, budget, family appeal, infrastructure, coastal lifestyle, and yield.
- Shows suburb, state, price band, rationale, and growth drivers for each suggestion.
- Works as a static site with no build step.

## Run locally

Open `index.html` directly in a browser, or serve the folder with a simple static server:

```bash
python3 -m http.server 8000
```

Then visit <http://localhost:8000>.
